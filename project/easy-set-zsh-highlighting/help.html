zsh-syntax-highlighting 高亮类型说明

unknown-token - 未知命令或语法错误，例如 curll http://google.com 中的 curll 会被标记。

reserved-word - Shell 保留字，例如 if、for。

alias - 普通别名，例如 alias ll='ls -l'。

suffix-alias - 后缀别名（需要 Zsh ≥5.1.1），例如 alias -s txt=vim，输入 file.txt 会自动执行 vim file.txt。

global-alias - 全局别名，可出现在命令任意位置，例如 alias -g G='| grep'，输入 ps aux G root 会展开为 ps aux | grep root。

builtin - Shell 内置命令，例如 pwd、shift。

function - 函数名。

command - 普通命令名（非别名、非内置）。

precommand - 命令前修饰符，例如 noglob、builtin。

commandseparator - 命令分隔符，例如 ;、&&。

hashed-command - Zsh hash table的命令(缓存在Zsh hash table中的命令),似乎已经失效,尝试在zsh中设置该选项,发现命令依旧为依旧为 command 的颜色。

autodirectory - AUTO_CD 开启时命令位置的目录名。

path - 已存在的文件名。

path_pathseparator - 文件路径分隔符（/），默认使用 path。

path_prefix - 已存在文件名前缀。

path_prefix_pathseparator - 前缀路径分隔符（/），默认使用 path_prefix。

globbing - 通配符表达式，例如 *.txt。

history-expansion - 历史命令展开，例如 !foo 或 ^foo^bar。

command-substitution - 命令替换，例如 $(echo foo)。

command-substitution-unquoted - 未加引号的命令替换。

command-substitution-quoted - 加引号的命令替换，例如 "$(echo foo)"。

command-substitution-delimiter - 命令替换分隔符 $( 和 )。

command-substitution-delimiter-unquoted - 未加引号的命令替换分隔符。

command-substitution-delimiter-quoted - 加引号的命令替换分隔符 "$( 和 )"。

process-substitution - 进程替换，例如 <(echo foo)。

process-substitution-delimiter - 进程替换分隔符 <( 和 )。

arithmetic-expansion - 算术展开，例如 $(( 42 ))。

single-hyphen-option - 单短横选项，例如 -o。

double-hyphen-option - 双短横选项，例如 --option。

back-quoted-argument - 反引号命令替换，例如 `foo`。

back-quoted-argument-unclosed - 未闭合反引号命令替换，例如 `foo。

back-quoted-argument-delimiter - 反引号分隔符 `。

single-quoted-argument - 单引号参数，例如 'foo'。

single-quoted-argument-unclosed - 未闭合单引号参数，例如 'foo。

double-quoted-argument - 双引号参数，例如 "foo"。

double-quoted-argument-unclosed - 未闭合双引号参数，例如 "foo。

dollar-quoted-argument - $'foo' 类型参数。

dollar-quoted-argument-unclosed - 未闭合 $'foo' 参数。

rc-quote - 单引号中两个连续单引号（RC_QUOTES 选项生效），例如 'foo''bar'。

dollar-double-quoted-argument - 双引号中的参数展开，例如 $foo。

back-double-quoted-argument - 双引号中反斜杠转义，例如 "foo\"bar"。

back-dollar-quoted-argument - $'...' 中的反斜杠转义，例如 $'\x48'。

assign - 参数赋值，例如 x=foo 或 x=(1 2 3)。

redirection - 重定向符号，例如 >、<。

comment - 注释（INTERACTIVE_COMMENTS 生效时），例如 echo # foo。

named-fd - 命名文件描述符，例如 {fd} 在 echo foo {fd}>&2。

numeric-fd - 数字文件描述符，例如 2 在 echo foo {fd}>&2。

arg0 - 除命令、函数、别名、内置命令外的命令字。

default - 其他未分类内容。
